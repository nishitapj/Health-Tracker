Health-Tracker

Health-Tracker is a simple web-based application that helps users record and monitor their daily health activities. The application allows users to input health data such as steps walked, water intake, and sleep hours, and visualize the information using charts and summaries. Data is stored securely in Firebase Firestore for persistence and real-time access.

Features

Add daily health entries (Date, Steps, Water, Sleep)

Store data in Firebase Firestore

View saved health records dynamically

Display interactive charts for trends analysis

Calculate weekly health summaries automatically

Easy navigation between Dashboard and Add Entry pages

Input validation for accurate data entry
